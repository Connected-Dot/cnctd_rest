# cnctd_rest

REST client wrapper with common HTTP patterns and JSON handling.

## Installation

```toml
[dependencies]
cnctd_rest = "0.1.6"
```

## Features

- Generic GET/POST with JSON serialization
- Multiple auth schemes (bearer, custom headers)
- Timeout support
